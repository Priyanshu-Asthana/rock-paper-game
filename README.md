# Rock-Paper-Scissors Game

Welcome to the Rock-Paper-Scissors game! This is a single-player game where you play against the computer. The game is built using HTML, CSS, and JavaScript.

## Features

- **Play Against Computer**: Choose rock, paper, or scissors and compete against the computer.
- **Interactive UI**: Simple and clean user interface with real-time results.
- **Instant Feedback**: See the result of each round immediately.

## Getting Started

### Prerequisites

- A web browser (Chrome, Firefox, Safari, etc.)

### Installation

1. **Open `index.html`**

   Open the `index.html` file in your web browser to start playing the game.

### How to Play

1. **Choose Your Move**

   - Click on the "Rock", "Paper", or "Scissors" buttons.

2. **See the Result**

   - The computer will make a random choice, and the result of the round (win, lose, or draw) will be displayed.

3. **Play Again**

   - Click on your choice again to start a new round.

## Files

- `index.html`: The main HTML file containing the structure of the game.
- `rockpaper.css`: The CSS file for styling the game interface.
- `rockapaper.js`: The JavaScript file containing the logic for game functionality.


## Troubleshooting

- **Game Doesn't Load**: Ensure you have the latest version of your web browser and try clearing the cache.
- **Buttons Not Working**: Verify that your browser's JavaScript is enabled.
